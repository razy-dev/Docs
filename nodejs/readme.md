# NodeJs
## Upgrade NodeJs & npm
---
### Unix/Linux
```console
]$ node -v
v8.11.4

]$ sudo npm cache clean -f
]$ sudo npm install -g n

]$ node -v

```

```console
]$ npm -v
5.6.0

]$ sudo npm install -g npm

]$ npm -v
```

### Windows
```console
> Set-ExecutionPolicy Unrestricted -Scope CurrentUser -Force
> npm install -g npm-windows-upgrade
> npm-windows-upgrade
```
